# FinancialModels Documentation

Welcome to the documentation home for the FinancialModels repository.

This documentation set is designed to do two jobs at once:

- read clearly on GitHub,
- map cleanly into a lightweight MkDocs site for GitHub Pages.

## What this project helps with

FinancialModels packages Codex skills that can:

- route a rough request into the right financial workflow,
- collect only the missing inputs,
- produce more structured Chinese finance deliverables.
