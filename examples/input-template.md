# Input Template

Use the following copy/paste templates when starting a model.

## DCF

```text
Company:
Industry / geography:
Revenue:
EBITDA or EBIT:
Capex:
D&A:
Tax rate:
Working capital behavior:
Net debt / net cash:
Share count:
5-year revenue growth assumptions:
5-year margin assumptions:
WACC inputs (if available):
Desired output: EV / equity value / per-share value
```

## Three-statement model

```text
Company:
Business description:
Latest income statement:
Latest balance sheet:
Latest cash flow statement:
Revenue growth assumptions:
Margin assumptions:
Capex assumptions:
Working-capital assumptions:
Debt / interest assumptions:
Forecast horizon:
```

## M&A accretion / dilution

```text
Acquirer:
Target:
Purchase price:
Cash / stock / debt mix:
Acquirer EPS or net income baseline:
Target earnings baseline:
Financing assumptions:
Synergies:
Desired output: EPS impact / pro forma bridge / recommendation
```

## LBO

```text
Company:
Industry:
LTM EBITDA:
Entry valuation:
Debt structure:
Sponsor equity:
Growth assumptions:
Margin assumptions:
Capex / tax / working-capital assumptions:
Exit year:
Exit multiple:
```

## Quick template for any model

```text
Model I want:
Company / deal:
What I want to know:
What data I already have:
What can be assumed if missing:
Preferred output language:
Preferred output style:
```
